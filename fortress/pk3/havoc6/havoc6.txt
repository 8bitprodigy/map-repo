* Other Info *

This map is designed to be played by two teams. Each team has there own
base. The objective is to enter the enemy base and hold their control
room.

  There is a group of monitors inside the control room which can be damaged
every 7 seconds. Each time any of them is damaged, the enemy team gets 1 point.

How to use this map:
Copy the havoc6.bsp file into your quake\id1\maps or quake\fortress\maps
directory. Then run quake.
In the console, type "map havoc" (without the quotes)

If you want to play on a QW server running this map, copy the havoc6.bsp
file into your quake\qw\maps directory.
