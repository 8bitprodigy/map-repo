* Play Information *

This map does not work properly without the TeamFortress v2.5 or 
higher QuakeC patch, which can be downloaded from 
http://www.planetquake.com/teamfortress or from ftp.cdrom.com

* Other Info *

How to use this map:

Client: Unzip duel1.bsp to your quake\fortress\maps directory. 
Server: If you dont know how, then you shouldn't have a server ;)
        If you're going to run this map all the time on a server,
        I suggest you run multiple servers on different ports, so
        more people can play at once. Share the fun :)
