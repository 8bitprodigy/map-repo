* Play Information *

This map does not work properly without the TeamFortress v2.5 or
higher QuakeC patch, which can be downloaded from
http://www.planetquake.com/teamfortress

Single Player           : Boring
Cooperative             : No
Deathmatch              : No
TeamFortress            : Very much so.  8 starting points
                          per team.
Difficulty Settings     : No
New Sounds              : No
New Graphics            : No
New Music               : No
Demos Replaced          : None

* Construction *

Base                    : New level from scratch
Editor(s) used          : WorldCraft v1.3, EntEd v2.0 Beta 2a
Known Bugs              : none yet...
Build Time              : Too long to release... :)
          (-extra) Light: ~10 minutes
          (-level 4) Vis: ~5 minutes
