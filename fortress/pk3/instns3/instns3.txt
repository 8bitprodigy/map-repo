How to use this map:
Copy the instns3.bsp file into your quake\id1\maps or quake\fortress\maps
directory. Then run quake.
In the console, type "map instns3" (without the quotes)

If you want to play on a QW server running this map, copy the instns3.bsp
file into your quake\qw\maps directory.
