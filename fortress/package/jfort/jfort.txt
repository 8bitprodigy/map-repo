* Other Info *

  
	Infultrate the enemy base and take thier runekey.
	Then take it up stairs and get thier Reactor Key.
	Take the Reactor Key and open the Reactor Room.
	After 5 captures, the Reactor Room will open, then
	destroy thier reactor with a detpack to end the level!  

  There are 2 resupply points for each team. One is on the second level
 level, and the other that can be reached by an under-water teleporter.


N.B. The grappling hook has been removed in this level.

How to use this map:
Copy the jfort.bsp file into your quake\fortress\maps
directory. Then run quake with the -game fortress.
In the console, type "map jfort" (without the quotes)

If you want to play on a QW server running this map, copy the jfort.bsp
file into your quake\qw\maps directory.
