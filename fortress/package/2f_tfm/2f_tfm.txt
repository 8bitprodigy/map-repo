 ======================= Installation ======================== 

 This archive contains 8 files.


  -2f_tfm.txt : 
   You are reading it now.

  -2f_tfm.bsp : 
   The map file. Place it with you other TeamFortress maps.

  -taken1, taken2, flagcap, reload, & teaser WAV's :
   These go in the quake/fortress/sound/tfm folder  

  -quad.mdl :
   This goes in the quake/fortress/progs folder
