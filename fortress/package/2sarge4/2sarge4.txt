* Other Info *
Description: 

Your basic 2 forts facing each other w/ a moat in middle and bridge. CTF style
scoring. Get the enemies flag and cap it on the rune texture of the balcony.

 
How to use this map:

Unpack 2sarge4.bsp to your quake\fortress\maps directory. If you want to run
it on a LAN server or just take a peek at it, load quake with -game fortress
and type MAP 2sarge4 in the console to load it. 

If you want to play on a QW server running this map, copy the 2sarge4.bsp
file into your quake\fortress\maps directory.
