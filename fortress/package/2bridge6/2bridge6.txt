*2bridge6*
Added the No grappling hook impulse, fixed a flag bug from who knows where, that kept
the team from picking up the flag when dropped.  This made gameplay and capture a very
uninspiring event... 
 
*2bridge5*
This revision fixed a info_tf bug, that would keep it from respwaning.

* Other Info *
If you like this level mail the server operators and let them know.

This is a large level.  2 fortresses with a bridge between the two.  

How to use this map:
Copy the 2bridge6.bsp file into your quake\id1\maps or quake\fortress\maps
directory. Then run quake.
In the console, type "map 2bridge6" (without the quotes)

If you want to play on a QW server running this map, copy the 2bridge6.bsp
file into your quake\qw\maps directory.
