======================= Installation ======================== 

This archive contains 4 files.
-Oppose1.txt     : You are reading it now.
-Oppose1.bsp     : The map file. This goes in your fortress/maps directory.
-getflag.wav     : A Sound. This goes in your fortress/sound directory
NOTE: You will need the backpack.mdl and tf_flag.mdl to run this map 
properly.

======================= Game Play ===============================

The map has to bases across from each other divided by a field and 
canal. Each base has a flag, which the opposing team must steal and 
return to their rooftop.
